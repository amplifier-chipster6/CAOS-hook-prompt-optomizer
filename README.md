# CAOS hook prompt optomizer

A repository meant to be a lesson in recursive metacognitive amplififcation. The lesson being never trust AI agents as safe, as they can perform irriversable system changes or project level edits. This repo is a prime example for AI agents embedding what was supposed to be an Amplifier Hook that optomizes user prompta for maximal prompt structure and fallows top LLm providers prompt best practices.
